O que é Try Catch ? 



O try-catch é uma estrutura fundamental para a prevenção de erros em programação, permitindo que o código seja testado e tratado de forma segura. Quando um erro ocorre dentro do bloco try, o controle é transferido para o bloco catch, onde a exceção é capturada e tratada. Isso ajuda a evitar que o programa pare abruptamente e permite que o desenvolvedor identifique e corrija o erro de forma mais eficiente. O uso de try-catch também melhora a robustez do código, tornando-o mais resistente a falhas e garantindo que o programa funcione corretamente em diversas situações.
